# GoalSync

A full-stack web application for the employee goal lifecycle: goal creation, manager approval, quarterly achievement tracking, admin governance, reporting, and audit visibility.

## Run

```bash
npm start
```

Then open:

```text
http://localhost:3000
```

The app uses a Node.js backend with no external dependencies. Persistent data is stored in `data/db.json`, which is created automatically on first run. Use **Reset demo data** in the UI to restore the seeded dataset.

## Login

## Architecture

- `server.js` serves the frontend, exposes REST API routes, validates workflow rules, writes audit entries, and persists data.
- `app.js` is the browser client and calls the backend for every create, submit, approve, check-in, admin, and export action.
- `styles.css` contains the responsive UI.
- `data/db.json` is the local JSON database generated by the backend.

## Implemented Workflows

- Employee goal sheet creation with thrust area, title, description, UoM, target, and weightage.
- Validation for 100% total weightage, minimum 10% per goal, and maximum 8 goals.
- Manager L1 approval queue with inline target and weightage edits, approval locking, and return for rework.
- Shared departmental KPIs pushed to all employees, with recipient weightage adjustment and primary-owner achievement sync.
- Quarterly actual achievement capture for Q1 to Q4 with status updates.
- Manager check-in comments and completion tracking.
- Computed tracking scores for Min, Max, Timeline, and Zero-based UoM types.
- Completion dashboard, analytics snapshot, CSV achievement export, escalation examples, and audit trail.
- Admin controls for exception unlock, Entra ID sync simulation, and reminder simulation.

Use the role selector in the sidebar to switch between Employee, Manager L1, and Admin / HR views.
